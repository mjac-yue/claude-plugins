## RICE Scoring

> **Note — RICE Scoring**: Makes implicit prioritization assumptions explicit. Key discipline: intellectual honesty about Confidence — overconfident estimates inflate scores and lead to building things that don't work as expected. Scores start the conversation; they don't end it.

> 💡 **Tip**: *[Your AI will flag any items where Confidence scores seem inconsistent with available evidence, and highlight which items have the most uncertain Effort estimates for your initiative.]*

| Feature / Initiative | Reach | Impact | Confidence | Effort (wks) | RICE Score | Rank |
|---------------------|-------|--------|------------|--------------|------------|------|
| | | | | | | |

**RICE Score** = (Reach × Impact × Confidence) / Effort

*Impact scale: 3=massive, 2=high, 1=medium, 0.5=low, 0.25=minimal*  
*Confidence: 100% / 80% / 50%*

## MoSCoW Categorization

> **Note — MoSCoW Categorization**: Adds a qualitative layer to RICE by forcing an explicit launch-blocking decision. Key discipline: the Must Have list should be the *smallest* set without which the product doesn't solve the core problem. When in doubt, demote to Should Have.
